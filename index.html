TakeoutFM

We are indie developers that build the free (as in freedom) software behind Takeout. We use open source cross-platform technology like Go, Flutter and Dart to support as many platforms as possible.

We focus on privacy, security, and freedom, with the overall goal to enable you to enjoy your personal media as you choose. There is no user tracking or phoning home with central servers.

All source code is available on our Github page. Software is distributed under terms of GNU licenses.
